# Hiedra - Perde Satış Uygulaması

Modern ve şık perde satış uygulaması.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm run dev
```

Uygulama http://localhost:3000 adresinde çalışacaktır.

## Özellikler

- Ürün listesi ve kategoriler
- Ürün detay sayfası
- Sepet işlemleri
- Modern ve responsive tasarım

## Teknolojiler

- React 18
- React Router DOM
- Vite
- Context API
